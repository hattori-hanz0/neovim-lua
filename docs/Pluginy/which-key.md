# which-key.nvim

Skrót wywołujący which-key: `\`

| Skrót | Opis                 |
| ----- | -------------------- |
| `b`   | Bufory               |
| `c`   | Colors               |
| `d`   | Dashboard            |
| `D`   | dot.files            |
| `f`   | Telescope find files |
| `g`   | Git                  |
| `k`   | Kolory               |
| `m`   | Dokumentacja         |
| `n`   | NvimTree             |
| `o`   | Window               |
| `q`   | Quit                 |
| `r`   | Telescope live grep  |
| `s`   | Search               |
| `t`   | Terminal             |
| `v`   | Neovim config        |
| `x`   | Save & Quit          |
| `z`   | Focus (ZenMode)      |

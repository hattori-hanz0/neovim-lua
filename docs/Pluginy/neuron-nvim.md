## Instalacja Neuron

- https://github.com/srid/neuron

```
curl -sfLo ~/bin/neuron.tgz https://github.com/srid/neuron/releases/download/1.0.1.0/neuron-1.0.1.0-linux.tar.gz
tar zxf ~/bin/neuron.tgz -C ~/bin
rm ~/bin/neuron.tgz
```
